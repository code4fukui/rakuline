# らくライン (rakuline)

<img src="rakuline.svg" width="200" alt="rakuline logo">

ハピラインふくい向けのシンプルなリアルタイム乗換案内アプリです。任意の2駅間の列車の時刻表、所要時間、および運賃を確認できます。

## デモ

**[https://code4fukui.github.io/rakuline/](https://code4fukui.github.io/rakuline/)**

![らくラインアプリのスクリーンショット](https://github.com/code4fukui/rakuline/assets/1715217/e456bfc8-f2cc-426b-b132-3635ce6c566a)

## 機能

-   **乗換案内**: 出発駅と到着駅を選択すると、次に乗車可能な列車を確認できます。
-   **リアルタイムカウントダウン**: 次の列車が出発するまでの残り時間を正確に表示します。
-   **詳細情報**: 各経路の出発・到着時刻、合計所要時間、および運賃を表示します。
-   **柔軟な日時選択**: 日時選択機能を使って将来の予定を立てるか、「現在時刻を反映」をクリックして現在時刻を基準にします。
